Entity Framework Core - .Set<> Method

Key Points:
  • The .Set<> method belongs to Entity Framework Core.
  • It is used to get a DbSet<> for the specified type during runtime.
  • This method allows you to dynamically determine the DbSet<> for a given entity type.
Example Code:

_context.Set().Add(user);
Explanation:
  • At runtime, the .Set<T>() method resolves to the corresponding entity type.
  • For example:
    • When T is replaced with User, the code becomes _context.Users.Add(user);.
  • This approach enables dynamic handling of entities without hardcoding the DbSet<>.
