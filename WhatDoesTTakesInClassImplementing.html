Code Explanation: IImplementingClassName<T>

Code:

public class IImplementingClassName<T> where T : class {}

Explanation:

  • The line '<T>' means that when you access the class methods, there will be a placeholder that you need to pass when creating an object (via Dependency Injection). You will pass the entity you want the method to work with, such as creating CRUD operations for a controller.
  • The line after '<T>', 'where T : class' means that '<T>' will only accept reference types. To make '<T>' accept value types instead, you would replace 'class' with the 'struct' keyword.
  • Alternatively, you can write this same code using 'is' instead of ':' The code will look like this:
    • 
public class IImplementingClassName<T> where T is class
