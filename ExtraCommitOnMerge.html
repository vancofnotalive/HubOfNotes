Git Branch Merging Notes

Merging a Feature Branch into the Main Branch

Whenever you merge a feature branch (e.g., feature/login) into the main branch (e.g., master or main), an additional commit is created.

Additional Commit for Merge Recording

This extra commit records the merge process, capturing details such as:

  • The branch name being merged.
  • The timestamp of the merge.
  • A list of merged changes.

Example 


# Assuming you are on the main branch
git checkout main

# Merging a feature branch
git merge feature/login

Summary

Git automatically creates a commit to log the merge process, ensuring a record is maintained for traceability and collaboration.
