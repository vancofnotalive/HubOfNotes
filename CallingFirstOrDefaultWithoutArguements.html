LINQ Method: FirstOrDefault

Code Example

var listOfNumbers = new List<int> {11 , 12 , 13 , 14 , 15}; var firstData = listOfNumbers.FirstOrDefault(); Console.WriteLine(firstData);

Output

11

Explanation

When you call the LINQ method .FirstOrDefault() without giving any arguments, it returns the first element of the collection it is called on. If the collection is empty, it returns the default value for the type of elements in the collection (e.g., null for reference types or 0 for integers).

Note: Using this method without arguments can sometimes cause a warning.
