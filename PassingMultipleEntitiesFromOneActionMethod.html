Passing View and JSON at the Same Time

Action Method: Return View and JSON

When you need to transfer both a view and a JSON object from an action method, you can use this approach:

            var data = new
            {
                history = userHistory,
            };
            return View("Index", System.Text.Json.JsonSerializer.Serialize(data));

In the View: Add Model Type

In the view, make sure to specify the model type as string to accommodate the serialized JSON:

@model string

Using JSON in JavaScript

If you need to pass the JSON as a parameter in a JavaScript function, use this script:

            var convertedJson = JSON.parse('@Html.Raw(Model)');
            gotJson(convertedJson);

Important Notes

Tip: Use @Html.Raw(Model) when you encounter issues with @Model in JSON.parse(). This ensures the JSON string is correctly rendered and avoids escaping issues.

In this case, the gotJson function is where you pass the JSON response for further processing.
